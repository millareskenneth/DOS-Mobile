Command Attribute Example
DIR - /A: Displays files with specified attributes.
- /S: Displays files in the specified directory and all subdirectories.
- /B: Uses bare format (no heading information or summary). 		C:\>dir /S /A:H
CD - /D: Changes the drive as well as the directory. C:\>cd /D D:\Documents
MD No attributes. md NewFolder (creates a new folder named NewFolder).
COPY - /Y: Suppresses prompting to confirm that you want to overwrite an existing destination file.
- /Z: Copies files in restartable mode. 		C:\>copy /Y file1.txt D:\Backup
DEL - /F: Force deletes read-only files.
- /Q: Quiet mode, do not ask if ok to delete on global wildcard. 		del file1.txt (deletes file1.txt).
REN No attributes. ren oldfile.txt newfile.txt (renames oldfile.txt to newfile.txt).
TYPE No attributes. type file.txt (displays the contents of file.txt).
CLS No attributes. cls (clears the screen).
DATE No attributes. date (displays the current date) or date 02-24-2024 (sets the date to February 24, 2024).
TIME No attributes. time (displays the current time) or time 14:30 (sets the time to 2:30 PM).
ECHO - /E: Enables command extensions.
- /S: Copies directories and subdirectories except empty ones. 		echo Hello, World! (displays "Hello, World!").
C:\>echo /E /S "Hello, World!"
FIND - /C: Displays only the count of lines containing the string.
- /I: Ignores the case of characters when searching for the string. 		find "keyword" file.txt (searches for "keyword" in file.txt).
C:\>find /C /I "keyword" file.txt
SORT No attributes. sort file.txt (sorts the contents of file.txt alphabetically).
PING - /T: Ping the specified host until stopped.
- /n: Number of echo requests to send. 		ping www.example.com (pings the website www.example.com).
C:\>ping /T /n 5 www.example.com
IPCONFIG - /ALL: Displays full TCP/IP configuration.
- /RELEASE: Releases the IP address for the specified adapter. 		ipconfig (displays network configuration information).
C:\>ipconfig /ALL
NETSTAT No attributes. netstat (displays active TCP connections).
TASKLIST - /V: Displays verbose task information.
- /SVC: Displays services in each process. 		tasklist (displays a list of running processes).
C:\>tasklist /V /SVC
TASKKILL - /F: Specifies to forcefully terminate the process(es).
- /IM: Specifies the image name of the process to be terminated. 		taskkill /IM notepad.exe (terminates the Notepad process).
C:\>taskkill /F /IM notepad.exe
FORMAT No attributes. format C: (formats the C: drive).
CHKDSK No attributes. chkdsk C: (checks the integrity of the C: drive).
TREE - /F: Displays the names of files in each directory.
- /A: Specifies how files should be displayed. 		tree /f (displays the directory structure with files).
C:\>tree /F /A
LABEL No attributes. label D: MyDisk (sets the volume label of the D: drive to MyDisk).
SHUTDOWN - /S: Shuts down the computer.
- /R: Restarts the computer. 		shutdown /s (shuts down the computer).
C:\>shutdown /S /T 0
ATTRIB - +: Sets an attribute.
- -: Clears an attribute. 		attrib +h file.txt (hides file.txt).
C:\>attrib +H file.txt
SET No specific attributes. set PATH=C:\NewFolder;%PATH% (adds C:\NewFolder to the PATH environment variable).
RMDIR No attributes. rmdir OldDirectory (removes the directory named OldDirectory).
MOVE No attributes. move file1.txt C:\NewLocation (moves file1.txt to C:\NewLocation).
DISKPART Specific attributes. diskpart (launches the DiskPart utility).
XCOPY - /S: Copies directories and subdirectories except empty ones.
- /E: Copies directories and subdirectories, including empty ones. 		xcopy /s folder1 C:\Backup (copies folder1 and all its subdirectories to the Backup directory).
C:\>xcopy /S /E folder1 C:\Backup
NET No specific attributes. net use X: \\server\share (maps the network share to drive X:).
